# Security Policy

## Supported Versions

The following table lists the versions of `p2p-media-loader` that are currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 2.x     | :white_check_mark: |
| 1.x     | :x:                |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, please report them using GitHub's private vulnerability reporting feature. You can do this by navigating to the **Security** tab of this repository and selecting **Report a vulnerability** under the **Advisories** section.

Alternatively, you can reach out directly to the maintainers at Novage via our [website](https://novage.com.ua/).

We will acknowledge receipt of your vulnerability report as soon as possible, and we will strive to send you regular updates about our progress. If the vulnerability is accepted, we will work with you to provide a timeline for a fix and public disclosure.
